About 31NS-Tech Hardware Lab

RF · BLE · NFC · Sensor · Mixed-Signal 하드웨어 전문 연구소

🔧 연구소 소개

31NS-Tech Hardware Lab은 RF · BLE · NFC · Sensor · Mixed-Signal 기반 회로 및 제품 개발을 전문으로 하는 엔지니어링 랩입니다.

지난 20년간 다양한 산업 환경 속에서 축적한 실제 경험을 바탕으로, 회로 설계, 펌웨어 협업, RF 튜닝, 양산 대응, 인증 대응까지 제품 개발의 모든 단계를 직접 수행하고 있습니다.

👨‍🔬 엔지니어 프로필

  • ✔ RF-BLE-NFC 기반 하드웨어 전문가
  • ✔ S전자, L전자 외 대기업 프로젝트 RF 설계 경험
  • ✔ Mixed-Signal Analog Front-End 개발
  • ✔ 반도체·디스플레이 생산라인용 모니터링 센서보드 개발
  • ✔ 다양한 전자회로 및 PCB설계
  • ✔ 양산제품 생산관리 및 공정문제 해결
  • ✔ 기능·성능·신뢰성·환경시험
  • ✔ KC·FCC·CE 인증부적합 대응

📡 주요 전문 분야

  • ✔ 2.4GHz ISM Band RF 회로 설계 및 최적화
  • ✔ Nordic nRFxx 기반 BLE 모듈 HW 개발 및 튜닝
  • ✔ 13.56MHz + 125kHz 복합 NFC 회로 설계
  • ✔ 임피던스 매칭 · 안테나 매칭 · RF 튜닝
  • ✔ Mixed-Signal 센서보드 개발
  • ✔ 고전압/전류 제어 회로 설계
  • ✔ 피에조 기반 96dB↑ 고출력 사운드 회로·구조 설계
  • ✔ 초음파 트랜스듀서 기반 콘크리트 경도·강도 측정 장비 개발
  • ✔ 동작감지 플래시·고출력 LED 플래시 회로
  • ✔ 초소형 빌드업 PCB 기반 다중 센서 AFE 통합 모듈 설계

🎯 연구소 미션

31NS-Tech Hardware Lab의 목표는 단순히 회로를 설계하는 것이 아니라, 고객이 가진 문제를 기술로 명확하게 해결하는 것입니다.

복잡한 회로 구조도 이해하기 쉽고, 측정 가능한 성능 개선으로 연결되도록 실전 중심의 접근 방식으로 프로젝트를 수행합니다.
